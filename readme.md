# siam [![Build Status](https://travis-ci.org/lvarayut/siam.svg?branch=master)](https://travis-ci.org/lvarayut/siam)

> Get your favorite Siam food

## Install

```
$ npm install --save siam
```


## Usage

```js
const siam = require('siam');

siam();
//=> 'Awesome Siam food: Tom yum'
```

### siam()


## CLI

```
$ npm install --global siam
```

```
$ siam --help

  Usage
    $ siam

  Examples
    $ siam
    Awesome Siam food: Tom yum
```
